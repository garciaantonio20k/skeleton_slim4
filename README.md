# Skeleton Slim 4
A basic structure for the [Slim PHP framework](http://www.slimframework.com/) version 4.*
For web application

## Installation

`composer create-project "garciantonio/skeleton_slim4" projectname`

## Contents

### Core
- slim/slim (4.10)
- slim/psr7
- php-di/php-di
### View
- slim/twig-view
- twig/twig (3.4.3)
### Env
- vlucas/phpdotenv
### Debug
- symfony/var-dumper
## Database
- PDO
